My Senior Project

"Let's Try Computer Sci"

By: Ethan Rosen

On my project, I will attempt to use Codecademy as a tool for learning the beginning of web development. I will work through their 14 course webdev and apply those new skills to building my own website through PyCharm CE, an application used for web development.

I chose to learn web development for my senior project because I had always had that idea that I wanted to start coding and building websites. I always had that intuition that it would be beneficial for college and for my career, though I never had a strong initiative to embark on learning it. Granted, I had no prior experience, so taking an app development class this year helped ease into the process a little more and will make starting web development a bit easier. I want to be able to use these skills in college and take computer science classes or help my peers learn these skills or with budding business ideas.

I hope to gain skills that I can apply in my college courses, in helping other teens build websites/learn comp sci., and to be a more attractive employment option when the time comes. Besides for those reasons, I was always fascinated by people discussing computer science and programming and I was never able to partake in that convo. I want to be able to learn skills to educate myself so that I can have those types of conversations. Additionally, I want to build websites and combine my business and programming affinities into a viable career or find jobs where I can combine the two.

I felt that, in order to best explain my progression through these past four weeks, I would add my Daily Logs and Weekly Reflections onto my site.

Please look below for my journey through Senior Project!
